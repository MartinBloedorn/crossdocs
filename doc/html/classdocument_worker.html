QStandardItemModel * documentWorker::getDocumentStructure ( )

Creates and returns the document's structure (sections, subsections, etc.) Uses the inputFileParser::parseInputFile() method to generate the structure for each input file. Applies this sequentially to each input file and then stitches those structures together, which generates the structure of the "bigger picture". Simultaneously updates the internal structural element list, that is later used for line number and index information checking.

Attention
This whole thing is based on the idea that ALL stuctural tags in a document are different. Violating this will generate unexpected behaviors!
Returns
The QStandardItemModel object containing the representation of the document's structure.
